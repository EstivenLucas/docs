## LARAVEL


## CREAR PROYECTO

```bash
 # 6.0.* como version
 $ composer create-project laravel/laravel nombre "6.0.*" --prefer-dist
```
## COMANDOS

```bash
 # Crear Controlador                                                               
 $ php artisan make:controller nombre                                
  
 # Crear Modelo 
 $ php artisan make:model nombre
   
 # Crear Provider
 $ php artisan make:provider nombre

 # Crear Middleware
 $  php artisan make:middleware {Nombre}

 # Lista de rutas creadas                                              
 $ php artisan route:list  
 ```
